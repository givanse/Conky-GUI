# Installation instructions

## tar.bz2

#### Install
Download ```conkyguiv#.#.tar.bz2``` and

    tar -jxvf conkyguiv#.#.tar.bz2

Then

    cd conkyguiv#.#
    sudo ./install

#### Remove

    conkygui-remove

## deb

#### Install

Download ```conkyguiv#.#.deb``` and

    sudo dpkg -i conkyguiv#.#.deb

#### Remove

    sudo apt-get remove conky-gui
