ConkyGUI

Conky GUI aims to quicken the customization of Conky files.

This is only a graphic user interface (GUI). If you don't have Conky yet, check the Conky website for the latest version.

Features:

  • syntax highlight
  • word completion
  • run conky from the gui
  • color pickers
  • easy access to configuration variables
